# 🌿 SUSTENTA — Plataforma de Consumo Consciente

O **SUSTENTA** é uma aplicação web desenvolvida para incentivar práticas sustentáveis no dia a dia, ajudando usuários a monitorar seu consumo, entender seu impacto ambiental e evoluir através de um sistema gamificado com educação e desafios.

A proposta combina **tecnologia, dados e conscientização** para transformar hábitos de consumo em ações mais responsáveis.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído com uma stack moderna e focada em performance e escalabilidade:

* ⚛️ React + Vite + TypeScript
* 🔥 Firebase (Authentication + Firestore)
* 🎨 Tailwind CSS
* 🎬 motion
* 🎯 lucide-react
* 📊 Recharts
* 🧠 D3.js + TopoJSON
* 🗺️ Google Maps JavaScript API
* 📄 jsPDF + html2canvas
* 📰 NewsAPI

## 📌 Funcionalidades Principais

### 📊 Dashboard Inteligente

* Visão geral do consumo
* Score de sustentabilidade
* Gráficos e alertas em tempo real

### 💸 Controle de Consumo

* Registro, edição e exclusão de gastos
* Classificação por categorias

### 📈 Relatórios Avançados

* Geração de relatórios em PDF
* Exportação de dados em CSV
* Comparação entre meses
* Gráficos de evolução

### 🌎 Mapa Interativo

* Visualização em tempo real
* Ecopontos próximos
* Rotas e distâncias
* Heatmap de dados ambientais

### 🎓 Educação Ambiental

* Notícias reais atualizadas
* Vídeos educativos
* Quiz com tempo e ranking global
* Calculadora de carbono com sugestões inteligentes
* Desafio diário com recompensas

### 👤 Perfil e Sistema Social

* Gerenciamento de perfil
* Sistema de grupos familiares
* Configurações de conta e tema (dark mode)

### 🔔 Notificações

* Sistema completo com filtros

## 🧠 Sistema de Gamificação

O SUSTENTA utiliza um sistema de **XP e níveis** para engajar o usuário:

### 🎯 Ganho de XP

* Registro de consumo
* Leitura de conteúdos
* Acertos em quizzes
* Participação em desafios

### 🏆 Progressão de Nível

O usuário evolui conforme acumula experiência:

> Iniciante → Consciente → Engajado → Ativo → Dedicado → Guardião → Protetor → Mestre → Guardião Verde → Eco Herói

## 🗂️ Estrutura do Projeto

Organização baseada em boas práticas de escalabilidade:

* `components/` → Componentes reutilizáveis
* `pages/` → Páginas principais da aplicação
* `contexts/` → Contextos globais (tema, notificações, etc.)
* `services/` → Integrações e lógica com backend
* `layout/` → Estrutura base da interface
